### Attributes

| 参数     | 说明  | 类型    | 默认值  | 必须    |
| ------- | ---- | ------ | ------- | ------ |
| data    | 提示数据列表 | Array | [] | Yes     |

### data

| 参数     | 说明  | 类型    | 默认值  | 必须    |
| ------- | ---- | ------ | ------- | ------ |
| type    | 类型（default/error/warning/info） | String | default | Yes     |
| text    | 显示的内容 | String | default | Yes     |
